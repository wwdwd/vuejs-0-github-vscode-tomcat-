关于 vue.js 发布程序到服务器研究

①进入项目根目录 ,输入 npm run build ; 进行打包。
②此时，会生成dist目录，存放打包后的程序。
③进入dist目录，输入，npm install -g http-server // 该命令只需执行一次, 安装过之后, 以后就不需要重复安装了，安装本地服务器。
④此时打开dist目录下的主页后，内容会空白，原因是没有找到资源文件，在打包前需要修改config/index.js配置文件
⑤在config/index.js中 找到 assetsPublicPath:     将内容修改为 './'
⑥此时可以访问打包后的主页。

上述安装的是本地服务器。
下面举例说明如何将项目发布到tomcat服务器上
①注意上面步骤的修改，打包出一个正确的dist文件夹。
②放在tomcat根目录下的 webapp文件夹下
②找到tomcat所在根目录，进入bin文件夹
③双击startup 程序 启动项目  
④访问localhost:8080/文件夹名 即可访问被打包项目
